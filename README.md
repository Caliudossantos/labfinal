#  Desafio Final: Construção de uma Pipeline DevOps Completa

## Visão Geral do Projeto

Este repositório documenta e contém o código para o **Desafio Final** da disciplina de DevOps, focado na construção de uma pipeline de CI/CD completa e robusta utilizando o poder das **GitHub Actions**. O projeto abrange todas as fases, desde a compilação do código-fonte até a implantação final em um provedor de nuvem.

###  Objetivo

Desenvolver uma solução de Integração Contínua (CI) e Entrega Contínua (CD) que automatize os processos de Build, Teste e Deploy, garantindo a qualidade e a agilidade na entrega da aplicação.
